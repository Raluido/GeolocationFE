Planteamiento e Instrucciones

La propuesta para ésta actividad consiste en la realización de una aplicación web que haga llamadas a una API REST para obtener y guardar datos de la base de datos y en la misma. Los datos con los que vamos a trabajar son geolocalizadores, puesto que vamos a trabajar con mapas virtuales a través de un plugin de javascript llamado Leaflet.

Por tanto, en mi caso, he utilizado el siguiente software:

  • Aplicación principal con Angular (Frontend)
  • Los estilos con material de Angular
  • Plugin en javascript Leaflet
  • Api REST de Agresta en CodeIgniter con Base de Datos Postgre
  • Api REST de Geoapify para obtener los datos de lat y lng a partir de una dirección
